# SciStream

SciStream is a clean and responsive science-news interface that showcases curated stories across major scientific fields such as Technology, Physics, NASA, Environment, and Health. Each article is displayed as a visually engaging card with images, headings, and accessible labels to ensure a smooth and inclusive browsing experience.

## Features

- **Responsive Grid Layout:** Content automatically adjusts to different screen sizes using modern CSS.
- **Accessible Markup:** Aria-labels and semantic HTML elements help screen readers interpret the page correctly.
- **Category-Based Cards:** Each article belongs to a clear science category (Tech, NASA, Physics, Environment, Health).
- **High-quality Imagery:** Eye-catching article thumbnails for an engaging reading experience.
- **Simple Navigation:** Each card links to a dedicated article page.

## Technologies Used

- HTML5  
- CSS3 (Grid, Flexbox, Responsive Design)  
- Google Fonts (Manrope, Source Code Pro)
